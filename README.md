## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
# devs-editfolio

A premium, highly interactive Neo-Brutalist portfolio tailored for professional video editors and creative directors.

## Features

- **Custom Lightbox Video Player**: Simplified, responsive, aspect-ratio-aware video viewer tailored to both horizontal and vertical videos.
- **Thematic Micro-animations**: Custom GSAP animations, including a dark-themed interactive preloader.
- **Local Assets Hosting**: Built-in optimization using locally hosted media assets for loading preloader screens and main showcase elements.
- **Neo-Brutalist Styling**: Clean high-contrast typography, bold borders, flat drop shadows, and minimalist UI modules.

## Tech Stack

- **Framework**: React / TypeScript
- **Bundler**: Vite
- **Styling**: Tailwind CSS / Vanilla CSS
- **Animation**: GSAP (GreenSock Animation Platform)
- **Icons**: Lucide React
